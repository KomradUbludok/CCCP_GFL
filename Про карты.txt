
11:57:40Статья о оздании карт
Далеко не просто создать ландшафт для будущей миссий...
1.Создайте папку в игре с названием вашей карты (не забудте про .rte в конце)
2.Потом создайте текстовый фаил index.ini
3.Введите туда следующее:
DataModule
ModuleName = название карты
IncludeFile = название карты.rte/terran.ini
4.Создайте terran.ini и введите следующее:
/////////////////////////////////////////////////////////
// Название мапы
AddScene = Scene
PresetName = Имя мапы
LocationOnPlanet = Vector(Внизу ставте расположение планеты)
X = число
Y = число
// Gravity acceleration, m/s^2
GlobalAcceleration = Vector
X = 0
Y = 20

// Terrain SceneLayer
Terrain = SLTerrain
PresetName = имя карты Terrain
BitmapFile = ContentFile(Создайте потом папку Scenes)
Path = имя карты.rte/Scenes/имя карты.bmp
WrapX = 0
WrapY = 0
ScrollRatio = Vector(Это показывает хотите ли вы зацикленную карту,если да X = 0)
X = 1
Y = 1
BackgroundTexture = ContentFile
Path = Base.rte/Scenes/Textures/DirtDark.bmp

/////////////////////////////////////////
// Material frostings

AddTerrainFrosting = TerrainFrosting
TargetMaterial = Material
CopyOf = Topsoil
FrostingMaterial = Material
CopyOf = Grass
MinThickness = 5
MaxThickness = 7
InAirOnly = 1

/////////////////////////////////////////
// Terrain Debris which is scattered

AddTerrainDebris = TerrainDebris
CopyOf = Gold

AddTerrainDebris = TerrainDebris
CopyOf = Plants

AddTerrainDebris = TerrainDebris
CopyOf = Boulders

// Background scene layer definitions
AddBackgroundLayer = SceneLayer
CopyOf = Near Layer
AddBackgroundLayer = SceneLayer
CopyOf = Clouds Layer
AddBackgroundLayer = SceneLayer
CopyOf = Sky Layer

5.В Scenes создайте картинку ландшафта(посмотрите у других карт и поймете как делать)
6.В CC делайте мапу(актеры,бункеры итд),сохраняйте(обязательно как предложит) и выходите
7. В папке .rte создастся фаил Scenes.ini- удаляйте!
8.Дописывайте в индексе :
IncludeFile = имя мапыr.rte/Scenes/имя мапы.ini
9.Теперь ничего вылетать и стираться не будет!
